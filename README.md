# scala-web-gradle-play-secure-scrypt-encode-simple

## Description
A call to playframework web
with simple login forms scrypt hash
with no encryption.

## Tech stack
- scala
- gradle
  - play
  - scrypt

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
http://localhost
- username: user
- password: pass

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
